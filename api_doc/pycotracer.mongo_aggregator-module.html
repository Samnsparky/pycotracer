Module mongo_aggregator

source code

Simple Mongo-DB serialization convienence module for pycotracer.

Convienence / reference implementation for serializing data read from the pycotracer micro-library to a database. The pycotracer micro-library provides programatic access to Colorado Transparency in Contribution and Expenditure Reporting Campaign Finance system data (TRACER). This is an unofficial library / access layer.

manage_id_reassignment(entry, reassign_id, force_copy)

source code 

Setting the _id value on an entity if appropriate.

Determines if the _id value on an entity should be set and if a shallow copy of that entity should be created if the _id property must be set. If reassign_id is False, this is a no-op regardless of the value passed for force_copy. Otherwise, _id is set to the value at the key 'RecordID' in the provided entry. If force_copy, a shallow copy will be made of entry before adding / updating the _id value.

Parameters:
  • entry (dict) - The entry to modify.
  • reassign_id (bool) - Flag to indicate if the _id value of entry should be set to the value at entry['RecordID'].
  • force_copy (bool) - Flag to indicate if the entry should be shallow copied before making any changes. No copy shall be made if no operations on entry will be run.
Returns: dict
The modified entry. Will return the shallow copy if a change was made to entry and force_copy was True.

source code 

Update a record of a contribution report in the provided database.

Parameters:
  • database - The MongoDB database to operate on. The contributions collection will be used from this database.
  • entry (dict) - The entry to insert into the database, updating the entry with the same _id if one exists.
  • reassign_id (bool) - Indicates if the entry's _id value should be set to RecordID value. Defaults to True.
  • force_copy (bool) - Indicates if the entry should be shallow copied before being modified. If False, the passed entry and code using it may experience side-effects. Setting to True can increase memory usage. Defaults to False.
  • db (pymongo.database.Database)
